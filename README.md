<h2 id="overview">Overview</h2>
<p>This project predicts diseases based on patient-reported symptoms using machine learning. 
It handles imbalanced data, visualizes disease distributions, and evaluates model performance 
with metrics like accuracy and confusion matrix.</p>

<h2 id="features">Features</h2>
<ul>
  <li>Symptom-based disease prediction</li>
  <li>Handles imbalanced datasets</li>
  <li>Visualizations for symptom and disease analysis</li>
  <li>Confusion matrix and performance metrics</li>
</ul>

<h2 id="tech-stack">Tech Stack</h2>
<ul>
  <li>Python</li>
  <li>Pandas, NumPy</li>
  <li>Scikit-learn, Imbalanced-learn</li>
  <li>Matplotlib, Seaborn</li>
</ul>

<h2 id="usage">Usage</h2>
<ol>
  <li>Load the dataset</li>
